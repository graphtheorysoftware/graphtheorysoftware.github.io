Topological Indices

The topological indices which are implemented in GraphTea as reports, are as follows,

 

  • First Zagreb Index: see [1]
  • Second Zagreb Index: see [1]
  • AutographiX Conjecture: See [2]
  • First Variable Zagreb Index: see [3]
  • First General Zagreb Index: see [4]
  • Second General Zagreb Index: see [5]
  • First Zagreb CoIndex: see [6, 7]
  • Second Zagreb CoIndex: see [6, 7]
  • First Reformulated Zagreb Index: see [8]
  • Second Reformulated Zagreb Index: see [8]
